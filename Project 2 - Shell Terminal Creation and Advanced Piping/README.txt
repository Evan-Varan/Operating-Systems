Run shhh.c as executable (./shhh) to run the shhh terminal.
input.txt is an included text file with input for pre
output.txt is an included text file that can be written to
sort.txt is an included text file with input for sort

shhh input is the same as the assignment file, some example input:
ls
ls -t -al
cat output.txt
ls | more | wc
./pre < input.txt | ./sort > output.txt
